# Experiments Log

An informal, chronological record of what was tried and what happened. Include failed experiments — negative results save future effort.

---

## Template

```
### YYYY-MM-DD — [Short title]

**Hypothesis**: What did you expect?

**Setup**: Model, hyperparams, data split, eval metric.

**Result**: What actually happened? Include numbers.

**Takeaway**: What does this mean? Should we pursue this direction?
```
