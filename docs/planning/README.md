# Planning

This is a free-form, iterative planning area. Documents here are living — update them as the project evolves.

| File | Purpose |
|---|---|
| [roadmap.md](roadmap.md) | Phase-by-phase milestone tracker |
| [experiments-log.md](experiments-log.md) | Informal log of what was tried and what happened |

## Conventions

- **roadmap.md** tracks *what we plan to do* and at what phase.
- **experiments-log.md** tracks *what we actually tried* — failed experiments included. Negative results are as valuable as positive ones.
- Feel free to add new files here (e.g., `sprint-2025-Q2.md`, `model-comparison-notes.md`). Keep the index in this README updated.
